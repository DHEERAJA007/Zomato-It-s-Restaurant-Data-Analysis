
# Zomato Restaurant Data Analysis

This project analyzes restaurant data inspired by food delivery platforms like Swiggy and Zomato.

## Tools
- Python
- Pandas
- Seaborn
- Matplotlib
- SQL (SQLite)
- Jupyter Notebook

## Project Structure

zomato-restaurant-analysis
│
├── data
│   └── zomato.csv
│
├── notebooks
│   └── zomato_analysis.ipynb
│
├── outputs
│   └── charts
│
├── requirements.txt
└── README.md

## Run

pip install -r requirements.txt

Open the notebook:

jupyter notebook notebooks/zomato_analysis.ipynb
